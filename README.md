## :construction_worker: What I am building
I build backend and platform systems where **correctness, auditability, and scalability** matter.

- **[FlowRecon](https://github.com/MarvinKweyu/flow-reconciliation)**  
  A **plugin-based, multi-tenant financial reconciliation engine** for matching invoices against bank transactions at scale.  
  Focus areas include deterministic scoring, idempotent pipelines, and tenant isolation.

- **[TambuaShamba](https://marvinkweyu.net/products)**  
  A **geospatial informatics platform** for monitoring soil organic carbon using spatial analytics in Kenya.

- **[Moneyup](https://marvinkweyu.net/products)**  
  A **local-first, cross-platform invoice management system** exploring offline-capable financial workflows.

- **[ColorDetect](https://pypi.org/project/ColorDetect/)**  
  Image and color detection algorithms used in computer vision workflows.

Additional projects include **Masomo / AssessMe** (modular education platforms), **SocialFeed**, and **Marastore**.
